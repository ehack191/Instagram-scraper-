# 🚀 IG-Leads-Refinery: Multi-Agent Instagram Intelligence Scraper

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)

**IG-Leads-Refinery** is an autonomous agentic pipeline designed to transform raw Instagram data into high-conversion sales leads. It uses multi-agent orchestration to bypass traditional scraping bottlenecks like data noise, formatting errors, and low-context outreach.

---

## 🧠 The "Agentic" Difference
Unlike standard scrapers that just output a CSV of usernames, this project uses **Long-Chain Reasoning** to:
1.  **Extract:** Systematically scrape profiles based on hashtags, locations, or followers.
2.  **Refine:** An AI agent cleans "bio noise" and extracts hidden contact info (emails/links).
3.  **Score:** A reasoning agent evaluates lead quality (0-100) based on engagement rates and niche relevance.
4.  **Personalize:** Generates custom DM/Email hooks using context-aware reasoning.

---

## 🛠️ Tech Stack
* **Engine:** Python / Playwright (Headless automation)
* **Orchestration:** Multi-agent logic (via Gemini / Custom Frameworks)
* **LLM Integration:** Gemini Series / Claude 3.5 (for lead scoring & cleaning)
* **Data Handling:** Pandas / SQLAlchemy
* **Anti-Detection:** Rotating Proxies & Session Fingerprinting

---

## 📋 Features
* ✅ **Deep Bio Extraction:** Pulls public emails, phone numbers, and external links.
* ✅ **AI Lead Scoring:** Filters out bots and low-engagement accounts automatically.
* ✅ **Recursive Data Cleaning:** Uses AI to standardize company names and job titles from unstructured bios.
* ✅ **Outreach Formatting:** Ready-to-use JSON/CSV formats for Mass DM or CRM import.
* ✅ **Rate-Limit Intelligence:** Built-in sleep variance to mimic human behavior.
